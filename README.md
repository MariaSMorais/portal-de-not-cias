# Portal de Notícias

Projeto desenvolvido durante os estudos de HTML5 e CSS3 com o objetivo de praticar a construção de layouts complexos utilizando CSS Grid, organização modular dos estilos e aplicação de boas práticas de desenvolvimento front-end.

## Acesse o projeto
https://mariasmorais.github.io/portal-de-noticias/

## Sobre o projeto

O projeto consiste em uma página inicial de um portal de notícias sobre tecnologia, organizada em diferentes seções para destacar conteúdos de maneira clara e moderna.

## A interface apresenta:

- Notícias em destaque
- Artigos mais lidos da semana
- Seção dedicada à Inteligência Artificial
- Conteúdos recomendados na barra lateral
- Navegação por categorias de tecnologia

O desenvolvimento teve como foco a utilização de HTML semântico, CSS Grid para estruturação do layout, reutilização de componentes e organização modular dos arquivos CSS.

## Tecnologias utilizadas:
- HTML5
- CSS3
- CSS Grid
- Flexbox
- Variáveis CSS
- Pseudo-classes (:hover e :has)
- Pseudo-elementos (::before)
- Organização modular dos arquivos CSS

## Funcionalidades:
- Layout responsivo organizado com CSS Grid
- Destaque para notícias principais utilizando sobreposição de conteúdo nas imagens
- Seções organizadas por categorias de notícias
- Efeitos visuais de interação em links e botões
- Utilização de variáveis CSS para padronização do projeto
- Estrutura modular dos estilos em múltiplos arquivos CSS

## Como executar o projeto:

Utilize o VS Code para executar o projeto utilizando a extensão Live Server.
